# Personal Portfolio Website (HTML + CSS)

## Project Overview
This is a personal portfolio website created using HTML and CSS. 
This project is part of my CSS task where I styled my existing HTML portfolio 
to make it visually better, well-designed, and user-friendly.

## Objectives
- Connect external CSS file to HTML
- Improve layout, colors, spacing, and design
- Use different CSS selectors
- Add hover effects
- Make the website responsive

## Technologies Used
- HTML5
- CSS3

## Folder Structure
index.html  
style.css  
README.md  

## How to Run
1. Download or clone the repository
2. Make sure index.html and style.css are in the same folder
3. Open `index.html` in any web browser

## Features
- Clean and simple UI
- Different CSS selectors used
- Hover effects on buttons/links
- Responsive design using media queries
- Proper sectioning and structure
